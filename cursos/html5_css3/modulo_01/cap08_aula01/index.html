Semântica no HTML5

- O HTML5, atualmente ele é focado apenas na semântica, não mais na forma.
- Ele não estiliza mais o site, não muda cor, fonte, as coisas apenas visuais.
- Toda a parte de estilizar, ou da forma, foi deixado para o CSS

TAGS obsoletas

- Existem algumas TAGS que vieram dos outros HTML, que estilizam a pagina
- Porém elas se tornaram OBSOLETAS, com o passar do tempo houve um separação
- É sempre recomendado, olhar as TAGS obsoletas na documentação oficial.
