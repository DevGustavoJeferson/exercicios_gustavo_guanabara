Border style

Existem algumas variações de estilos de bordas, são eles

  • solid - mais usada, basicamente uma linha
  • dashed - pontos mais grosseiros
  • dotted - pontinhos
  • double - duas linhas
  • outset - uma linha meio 3d em diagonal
  • ridge - um linha 3d completa

Padding

O padding é o espaço entre o conteúdo e a borda, podemos utiliza-lo para melhor controle do nosso conteúdo

Quando formos adicionar o padding precisamos seguir uma ordem de primeiro cima, depois direita, baixo e esquerda
