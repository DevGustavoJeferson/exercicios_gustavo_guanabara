Pseudoclasses

São relacionadas por uma tag, ou uma classe

Elas basicamente dão uma certa função para a classe ou tag selecionada

Sua sintaxe segue basicamente:

element:function

O elemento vai ser afetado pela função, os dois pontos indica uma pseudoclasse

Algumas funções:

    • hover - quando passar o mouse em cima haverá alguma mudança

    • Child

    Uma tag dentro de outra é uma relação de hierarquia de pai e filho, essa hierarquia podemos personalizar no CSS.

    Sua sintaxe é:
    Pseudoclass > tagChild
