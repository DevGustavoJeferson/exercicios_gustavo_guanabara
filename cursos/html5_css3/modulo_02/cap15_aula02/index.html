Id x Classe

Qual a diferença entra id e classes?

O id é para um elemento separado, já uma classe poderá aparecer varias vezes

Classe

A classe pode ser utilizada em mais de uma tag

Sua sintaxe é: "class='nameClass'"

Não der nome pelas caracteristicas do CSS, e sim pela sua função dentro do projeto

Herança de classes

Assim como OOP, podemos fazer uma classe que herde atributos de outro classe

Bastando apenas colocar o nome da classe já criada e a subclasse separada por espaço

Id e Classes

Id sobrepoe classe, tendo assim uma ligação de herança
